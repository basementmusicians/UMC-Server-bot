# Commands
###### Utilities Bot
- `+help` - Shows the help command for the bot.
- `+ping` - Shows the bot latency.

------------

###### Management Bot
- `-help` - Shows the help command for the bot.
- `-ping` - Shows the bot latency.

------------

###### Production Bot
- `%help` - Shows the help command for the bot.
- `%ping` - Shows the bot latency.
- `%tips` - Returns some music production tips.

------------

###### Radio Bot
- `~help` - Shows the help command for the bot.
- `~ping` - Shows the bot latency.

------------

###### Chatbot Bot
- `=help` - Shows the help command for the bot.
- `=ping` - Shows the bot latency.
